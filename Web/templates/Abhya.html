Hello! I am Abhya Rabjan Yadav, student of BML Munjal University Batch-2015.

The part of project made by me includes channel page for the website that contains a base model that is common in all the webpages. The channel page will include a picture for the channel with channel’s name. There is an upload button on the right side of the picture. The upload button helps the user to upload the video on his channel. In the next line, there is a description about channel .Now, next line contains videos which are represented in the grid format using class row that contains four columns. These rows were repeated using for loop in django. Each cell contains an image in a division and another div contains texts like song’s name and artist’s name at the bottom of the image. Another page made by me is upload video page in which user can get a form in which he has to fill the details of the video like song name, artist name, and description of the video, duration and genre. The duration can be entered in minutes and seconds. The genre include Punjabi, Bollywood, and Hollywood songs. To upload the video, user can browse his computer for the desired video or there is an option to drag and drop the video. At last, there is a submit button that will finally upload your video with all the details that user has entered.
